#+TITLE: LLM Module
#+DATE: December 25, 2025

Agent-shell and ACP integration for LLM interactions.

* Description
This module provides integration with agent-shell and ACP (Agent Chat Protocol),
enabling direct LLM agent interactions within Emacs through a shell-maker based interface.

This is a minimal module focused on loading the required dependencies for agent-based
LLM interactions.

* Packages
The module loads three interconnected packages:

** shell-maker
Base infrastructure for creating shell-like REPL interfaces in Emacs.

** acp
Agent Chat Protocol implementation for structured agent communications.

** agent-shell
Shell interface for interacting with LLM agents, built on shell-maker and ACP.

* Installation
Enable the module in your =init.el=:

#+begin_src emacs-lisp
(doom! :offby1
       llm)
#+end_src

Then run:
#+begin_src shell
doom sync
#+end_src
